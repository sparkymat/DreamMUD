# DreamMUD

DreamMUD is a MUD codebase inspired by ResortMUD, and developed in Ruby.

## Installation

Add this line to your application's Gemfile:

    gem 'dream_mud'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install dream_mud
